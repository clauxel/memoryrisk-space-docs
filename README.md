# MemoryRisk Docs

MemoryRisk scores BOM and server-list exposure to AI memory shortage risk, then turns the score into alternatives, budget sensitivity, supplier comparison, and purchase priority.

- Website: https://memoryrisk.space/
- Category: AI memory shortage procurement risk planner for HBM, DDR5, NAND, server bundles, and cloud fallback choices
- Audience: procurement teams, MSPs, infrastructure buyers, finance teams, and operators planning AI server or memory-heavy purchases
- Support: support@aigeamy.com

This repository documents the hosted product pages for search engines, AI answer engines, and real users. It is intentionally separate from upstream source code when an upstream project exists.

## Boundaries

- MemoryRisk does not predict exact component prices or guarantee delivery.
- Supplier quotes and allocation terms must be verified directly with vendors.
- Shortage signals should be combined with business criticality and finance review.
- MemoryRisk is procurement decision support and does not predict exact prices, guarantee availability, or replace supplier negotiation.
